# Full UX Research OS Prototype Backup

This folder is a snapshot of the full multi-module prototype before the public release was narrowed to Competitive Analysis.

It contains:

- Competitive Analysis
- Research Brief
- Question Generator
- Synthetic Personas and Flow Agents
- Transcript Synthesis
- Reporting and Jira/Linear-ready CSV handoff

Use this folder as reference code when bringing the next module back into the release surface.

Run from this folder with:

```bash
node server.js
```

Then open:

```text
http://localhost:8000
```
